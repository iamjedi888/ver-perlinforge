★ TriptokForge TCG · Development Roadmap

TriptokCard

Epic Games player data transformed into playable trading cards.
MTG mechanics × Pokémon type system × Fortnite live stats — all converging into one competitive card game.

Vision

Generate unique, data-driven trading cards from Epic Games player statistics. Every player's performance metrics — wins, eliminations, K/D ratio, playstyle — feed directly into card rarity, power level, element type, and special abilities. Your Fortnite career becomes a deck.

Development Roadmap

Phase 1 — Card Generation Engine

Planned
  • Connect to Epic Games API for live player stats
  • Map stats to card attributes: Wins → Rarity, Eliminations → Power, K/D → Toughness
  • Dynamic card art generation using player skin + background palette
  • Ability system derived from player achievements and playstyle

Phase 2 — Card Browser & Deck Builder

Planned
  • Searchable card database with filters (rarity, element, power range)
  • Drag-and-drop deck builder with 60-card minimum enforcement
  • Mana curve visualization and synergy analyzer
  • Save / load decks to Oracle DB per account

Phase 3 — Game Engine

Planned
  • Turn-based gameplay: Draw → Main → Combat → End phase loop
  • Type advantage matrix (Fire > Grass > Water > Fire, etc.)
  • Combat resolution with MTG-style stack mechanics and instants
  • AI opponent for practice mode with adjustable difficulty

Phase 4 — Multiplayer & Social

Planned
  • Real-time PvP battles via WebSockets
  • Ranked ladder system with seasonal resets
  • Tournament brackets with prize pools
  • Spectator mode and replay system
Core Mechanics

⚡ Element System

Six elemental types — Fire, Water, Grass, Electric, Dark, Psychic — assigned based on your Fortnite playstyle and weapon preferences.

⚔ MTG Combat

Cards are Creatures, Instants, Sorceries, Enchantments, or Lands. Stack-based resolution allows for counterplay and reactive strategies.

📊 Stat Conversion

Wins → Rarity tier  |  Eliminations → Power  |  K/D → Toughness  |  Win Rate → Special ability threshold

🏆 Rarity Tiers

Common, Uncommon, Rare, Epic, Legendary, Mythic — determined by cumulative player performance over the season.

🌐 Social Deck Sharing

Share deck codes with the community. Browse top-rated decks in a public gallery. Import decks directly into your deck builder.

🔗 Live Data Sync

Cards update seasonally as Fortnite stats evolve. A player who levels up gets boosted card stats at each season checkpoint.
