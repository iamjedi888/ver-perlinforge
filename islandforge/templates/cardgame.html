🎴 TRIPTOKFORGE TCG

Epic Games Player Stats → Playable Trading Cards
MTG Mechanics × Pokemon Types × Fortnite Data

Vision

Generate unique trading cards from Epic Games player statistics. Each player's performance becomes a playable card with stats, abilities, and elemental types derived from their gameplay style.

Development Roadmap

Phase 1: Card Generation Engine Planned

  • Connect to Epic Games API for player stats
  • Map stats to card attributes (Wins → Rarity, Eliminations → Power, etc.)
  • Dynamic card art generation
  • Ability system based on achievements

Phase 2: Card Browser & Deck Builder Planned

  • Searchable card database
  • Drag-and-drop deck builder (60 card minimum)
  • Mana curve visualization
  • Save/load decks to Oracle DB

Phase 3: Game Engine Planned

  • Turn-based gameplay (Draw → Main → Combat → End)
  • Type advantages (Fire > Grass, Water > Fire)
  • Combat resolution with MTG stack mechanics
  • AI opponent for practice

Phase 4: Multiplayer & Social Planned

  • Real-time PvP battles via WebSockets
  • Ranked ladder system
  • Tournament brackets
  • Spectator mode
