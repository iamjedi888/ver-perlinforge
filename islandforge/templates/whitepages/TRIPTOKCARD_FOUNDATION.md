# TriptokCard Foundation

TriptokCard is the original tactical card-game lane for TriptokForge. It is not a Magic or Pokemon clone. The goal is to borrow the depth of strong TCG design while keeping the rules, art language, names, and role system original to the site.

## Core Roles

- `Leader`: named deck anchor and identity core
- `General`: command support piece that amplifies formations
- `Unit`: board-presence card for lane pressure and defense
- `Tactic`: instant-speed reaction or burst action
- `Field`: persistent arena modifier
- `Relic`: artifact-grade passive or trigger object
- `Primordial`: late-game mythic closer

## Identity Layer

- `Essence` controls elemental mood and interaction style
- `Gene` controls lineage identity and synergy behavior
- `Archetype` controls how the shell curves and spends tempo

The live prototype currently maps player and site stats into:

1. `Leader`
2. `General`
3. `Unit`
4. `Tactic`
5. `Field`
6. `Relic`
7. `Primordial`

## Turn Flow

1. Draw and refresh charge
2. Deploy Leader or Unit alignment
3. Open Command window for General / Field / Relic
4. Enter Clash
5. Use Tactics during reaction and resolve windows
6. Stabilize board state and end round

## Build Intent

- Original frame language with Triptok cyber-lab styling
- Role-based deck shell instead of copied card taxonomy
- Readable depth first, then more competitive complexity
- A-Frame prototype now, rules engine and multiplayer later
