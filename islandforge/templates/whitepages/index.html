Platform Overview

Fortnite esports community — island generation, live channels, social feed, player profiles.

What it does

  • Epic Games OAuth2 authentication (brand approval pending)
  • UEFN island heightmaps generated from audio frequency analysis
  • 65-channel live TV guide across 18 categories
  • TikTok-style social feed for clip sharing
  • Oracle Autonomous Database via python-oracledb thin mode
  • Media via OCI Object Storage public bucket CDN
◈
Status

Live at triptokforge.org. Blueprint v4.0. Epic OAuth pending brand review — all other features operational.

System Architecture

Oracle Always Free ARM — nginx → Gunicorn → Flask Blueprints.

Infrastructure

  • VM: Oracle Always Free ARM — Ubuntu 22.04 @ 129.80.222.152
  • Proxy: nginx → Gunicorn :5000
  • App: Flask Blueprints, systemd islandforge.service
  • DB: Oracle Autonomous — tiktokdb_high DSN, wallet auth
  • Storage: OCI bucket triptokforge · us-ashburn-1
  • Repo: github.com/iamjedi888/ver-perlinforge · branch main

Blueprint Structure

  • routes/platform.py — home, forge, gallery, feed, community, admin, health
  • routes/api.py — /api/members, /api/post, /api/like
  • routes/channels.py — channels + suggest endpoint
  • routes/auth.py — Epic OAuth flow
  • routes/whitepages.py — dev docs

Deploy Guide

Commit locally → push GitHub → pull VM → restart.

sudo systemctl restart islandforge && curl http://127.0.0.1:5000/health

06

Verse Architecture

Tarkov-Lite systems in Verse for UEFN. Immutable player_data pattern throughout.

Core patterns

  • Immutable data: MakePlayerData(Data) — always construct new, never mutate
  • UI state: weak_map(player, canvas) per-player HUD
  • Material encoding: rarity * 18 + element → 108-slot array
  • Persistence: @persist at field type level
  • Death penalty: under 5 stacks = 100% loss; over 5 = 75% loss

Island Forge

Audio → heightmap pipeline for UEFN island generation.

01
Upload

file_compressor.py auto-compresses to 128k MP3 via ffmpeg.

02
Analysis

librosa extracts bass (mountains), mid (terrain), high (rivers).

03
Generation

Bass energy = elevation. Rhythm = variation. Tone = biome.

04
Export

PNG + metadata saved to OCI. URL written to island_saves table.

Tarkov-Lite Vision

Persistent extraction MMO on Fortnite — Tarkov risk/reward inside UEFN.

Safe town hub. Load into extraction zones to farm, kill, extract. Real death penalty. Insurance tokens. Leaderboards, cosmetics, farm plots, seasonal world evolution.

Economy loop

  • Farm seeds → grow materials → sell at vendor for coins/gold
  • Kill enemies → drops + XP + quest credit
  • Extract safely → bank your run. Die → lose your carry
  • Buy insurance tokens → protect one item per run
  • Level up → unlock higher zones and cosmetics
12

Game Systems

All systems in Verse, communicating via immutable player_data.

Death penalty

  • Under 5 stacks — 100% inventory loss on death
  • Over 5 stacks — 75% loss
  • Insurance tokens protect one item per run

Material system

  • 5 categories: Seeds, Fiber, Ore, Crystal, Mythic
  • 18 elements × 6 rarities = 108 slots — encoded as rarity * 18 + element
  • Value: Common ×1 → Mythic ×75
