# UE Trademark Submission

## Current official path

Epic's public Unreal branding page says:

- use the approved Unreal Engine logos from the branding page
- complete and return the `Trademark License Agreement`
- if you want to customize the Unreal Engine logo to fit your branding, submit example materials to `branding@unrealengine.com`
- accepted file formats for review include `JPEG`, `PNG`, `PDF`, and videos in `WMV` or `MOV`

The trademark agreement PDF also says the completed agreement should be submitted to:

- `trademark@epicgames.com`

## What this means for TriptokForge

If your Unreal Engine Trademark License Agreement is already accepted:

- standard, unmodified Unreal Engine logo use should follow the approved branding guide
- custom treatments, co-lockups, or logo presentations integrated into your site design should still be submitted as examples before you ship them

## Recommended submission packet

Send a concise package to `branding@unrealengine.com` with:

1. a short plain-language request
2. the product/site name: `TriptokForge`
3. your company/studio name
4. the exact URLs or page types where the mark will appear
5. 2-4 mockups in `PNG` or `PDF`
6. the exact Unreal mark asset used
7. whether the treatment is:
   - unchanged approved mark
   - layout integration only
   - custom/co-branded treatment
8. whether the use is:
   - website
   - login/auth page
   - marketing page
   - trailer/video
9. a line confirming you will not publish before written approval if the treatment is custom

## Suggested example set

For TriptokForge, the best review set would be:

- login/connect page with a small factual Unreal Engine mark
- WhitePages branding/compliance section
- one side-by-side brand treatment example
- one mobile version if the mark appears on mobile

## Important boundary

Public Epic sources do not provide a self-serve path for an official:

- `TriptokForge x Fortnite`
- `TriptokForge x Epic Games`
- `TriptokForge x Unreal Engine`

co-brand license beyond the standard Unreal trademark path.

So for any real crossover or co-branded lockup, assume you need separate written permission from Epic beyond the ordinary Unreal logo approval flow.

## Recommended ask for crossover

If you want an official crossover treatment, ask for it explicitly in writing:

- state that the Unreal trademark agreement is already accepted
- attach the mockups
- ask whether the proposed side-by-side or `x` treatment is permitted
- if not, ask which Epic team handles broader co-brand, licensing, or partner-brand review

Do not ship a crossover lockup unless Epic says yes in writing.
