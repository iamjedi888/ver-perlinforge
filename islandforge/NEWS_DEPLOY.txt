## API Endpoints:

- `/api/news/feeds` - List all feeds
- `/api/news/regions` - Feeds grouped by region
- `/api/news/feed/<feed_id>` - Specific feed
- `/api/news/latest` - Top 50 headlines (all regions)
- `/api/news/region/<region_name>` - Region-specific news

## Add More Feeds:

Edit `islandforge/routes/news.py`, find `NEWS_FEEDS` dict:

```python
"your_feed": {
    "name": "Feed Name",
    "url": "https://example.com/rss.xml",
    "region": "rhode_island",  # or japan, california, etc.
    "type": "rss",
    "category": "local"
}
```

Restart service to apply changes.

## Troubleshooting:

**News not loading:**
- Check feedparser installed: `pip3 list | grep feedparser`
- Test RSS URL manually: `curl <feed_url>`
- Check logs: `sudo journalctl -u islandforge -n 50`

**Some feeds fail:**
- Normal! Some RSS feeds go down temporarily
- The system skips failed feeds and shows working ones

**Ticker not scrolling:**
- Check browser console for JavaScript errors
- Verify news.js is loaded
